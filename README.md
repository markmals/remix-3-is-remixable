# Remix 3 is Remixable

Code examples from my November 2025 talk at the [Remix Austin Meetup](https://www.meetup.com/remix-austin/)

## Setup

Install [Mise](https://mise.jdx.dev/installing-mise.html) and run:

```sh
mise install
mise run dev
```
